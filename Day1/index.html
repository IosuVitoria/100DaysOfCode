EMAIL VALIDATION - FORMULARIO CON VALIDACIONES

Instrucciones - Instructions

For the first day, the task is to create a form with validations. The user will provide us with information, and before sending it to the server, we will verify it. If it's incorrect, a message will be displayed to the user.

Para el primer día se tiene que hacer un formulario en el cual se pida al usuario sus datos personales y un correo y se valide según se envía para comprobar en base de datos.

Registro de usuarios

Introduzca los datos correctamente para que podamos atender a sus peticiones.
