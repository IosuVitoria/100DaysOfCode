Secuencia Fibonacci

La secuencia de Fibonacci es una serie de números en la que cada número es la suma de los dos anteriores. Comienza con el 0 y con el 1.

Para calcular la secuencia, comenzamos con los dos primeros números (0 y 1), luego cada número subsiguiente es la suma de los dos anteriores.


Cómo funciona el código JavaScript:

El siguiente código JavaScript se ha preparado para generar y muestra los primeros n números en la secuencia de Fibonacci. Puedes leer las explicaciones intercaladas para entender su planteamiento:


            // Obtener referencia a los elementos del DOM. Se captura por una parte el formulario que 
            // es dónde se encuentra ubicado el evento submit y por otra parte el elemento dónde se 
            // mostrará el resultado de la petición.
            const form = document.getElementById('fibonacci-form');
            const sequenceOutput = document.getElementById('sequence-output');
        
            // Función para generar la secuencia de Fibonacci. Se arranca con la secuencia inicial conocida de [0,1]
            function generateFibonacciSequence(n) {
                const sequence = [0, 1];
                for (let i = 2; i < n; i++) {
                sequence[i] = sequence[i - 1] + sequence[i - 2];
            }
            return sequence;
            }
        
            // Escuchar el evento submit en el formulario. Como está capturado se puede escuchar el evento submit y
            // arrancar el funcionamiento del mismo.
            form.addEventListener('submit', (event) => {
            event.preventDefault(); //Evitar el funcionamiento por defecto del formulario. 
        
            const numberInput = document.getElementById('number-input'); // Se captura el número.
            const n = parseInt(numberInput.value);
        
            if (isNaN(n) || n <= 0) {
                alert('Por favor, ingresa un número positivo válido.'); //Capturar un posible error antes de 
                //que se produzca.
                return;
            }
        
            const fibonacciSequence = generateFibonacciSequence(n); // Se llama a la primera función que se ha generado para 
           // poder crear la secuencia.
        
            let output = 'Secuencia Fibonacci:'; //Mostrar la secuencia en forma de lista.
                fibonacciSequence.forEach(num => {
                output += `${num}`;
            });
            output += '';
        
            sequenceOutput.innerHTML = output;
            });
