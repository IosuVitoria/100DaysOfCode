Calculadora de Notas

Instrucciones de Uso

  1. Introduce el nombre del alumno en el campo "Nombre del Alumno".
  2. Para cada asignatura:
    • Ingresa el nombre de la asignatura en el campo correspondiente.
    • Ingresa la nota del alumno para cada trimestre.
  3. Puedes hacer clic en "Agregar Asignatura" para añadir más asignaturas.
  4. Cuando hayas ingresado todos los datos, haz clic en "Calcular Medias".
  5. La tabla mostrará las notas por trimestre, el promedio y la evaluación para cada asignatura.
